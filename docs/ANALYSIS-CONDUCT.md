# Monte Hair Clinic — Blood Test Analysis Conduct

## Priority System

### 🔴 Priority 1: Action Required (ต้องดำเนินการ)
| Test | Threshold | Significance |
|------|-----------|-------------|
| Hemoglobin (Hb) | < 10.0 g/dL | โลหิตจางรุนแรง / เสี่ยงต่อการผ่าตัด / Hold Procedure |
| Hematocrit (Hct) | < 30% | เสี่ยงภาวะแทรกซ้อนจากการเสียเลือด |
| Platelet | < 100k /mm³ | เสี่ยงเลือดหยุดยาก / ระงับผ่าตัด |
| Ferritin | < 30 ng/mL | ธาตุเหล็กต่ำวิกฤต / เสี่ยงผมร่วง TE แบบเฉียบพลัน |
| Vitamin D (25-OH) | < 20 ng/mL | พร่องวิตามินดีรุนแรง / ส่งผลต่อ Graft Survival Rate |
| TSH | < 0.1 or > 10.0 uIU/mL | ไทรอยด์ผิดปกติชัดเจน / พบแพทย์เฉพาะทาง |
| Free Testosterone | > 60.00 ng/dL | ฮอร์โมนเพศชายสูงเกินเกณฑ์มาก / เสี่ยง AGA ลุกลาม |
| DHEA-S | > 400 µg/dL | ฮอร์โมนสูงผิดปกติ / PCOS / ปรึกษาสูตินรีแพทย์ |
| ANA | Positive + อาการร่วม | ปรึกษาแพทย์อายุรกรรมโรคข้อ/ผิวหนัง |

### 🟡 Priority 2: Health Optimization (ควรปรับปรุงและเฝ้าระวัง)
| Test | Threshold | Significance |
|------|-----------|-------------|
| Hemoglobin (Hb) | 10.0–11.9 g/dL | โลหิตจางเล็กน้อย / ให้วิตามินเสริม / ทำหัตถการได้ |
| Hematocrit (Hct) | 30–35% | เลือดจางเล็กน้อย / เฝ้าระวังระหว่างผ่าตัด |
| Platelet | 100k–149k /mm³ | เกล็ดเลือดค่อนข้างต่ำ / เฝ้าระวัง Hematoma |
| Ferritin | 30–69.9 ng/mL | ไม่เพียงพอต่อการสร้างรากผมใหม่ / จ่ายยาเสริมธาตุเหล็ก |
| Vitamin D (25-OH) | 20–29.9 ng/mL | วิตามินดีต่ำ |
| TSH | 0.1–0.34 or 4.95–9.9 | เริ่มผิดปกติ / ติดตามอาการ |
| Free Testosterone | 53.36–60.00 ng/dL | ปริ่มเกณฑ์ / เฝ้าระวังอาการผมบาง |
| DHEA-S | 341–400 µg/dL | สูงเล็กน้อย / เฝ้าระวัง PCOS |
| ANA | Positive ไม่มีอาการ | ผลบวกลวงหรือยังไม่แสดงอาการ / เฝ้าระวัง |

### 🟢 Priority 3: Healthy Baseline (เกณฑ์ปกติ)
| Test | Range |
|------|-------|
| Hemoglobin (Hb) | 12.0–16.0 g/dL |
| Hematocrit (Hct) | 36–47% |
| Platelet | 150k–450k /mm³ |
| Ferritin | 70–204 ng/mL |
| Vitamin D (25-OH) | ≥ 30 ng/mL |
| TSH | 0.35–4.94 uIU/mL |
| Free Testosterone | 13.84–53.35 ng/dL |
| DHEA-S | 98.8–340 µg/dL |
| ANA | Negative |

## Clinical Significance

- **Hemoglobin**: ภาวะซีดทำให้ออกซิเจนไปเลี้ยงรากผมน้อยลง ผมร่วงและกราฟท์ติดยาก
- **Hematocrit**: สอดคล้องกับ Hb บ่งชี้ความเข้มข้นเม็ดเลือดแดง
- **Platelet**: ส่งผลต่อ Microcirculation บริเวณหนังศีรษะ
- **Ferritin**: ธาตุเหล็กจำเป็นต่อเซลล์รากผม ≥70 ng/mL สำหรับสุขภาพผม
- **Vitamin D**: มีผลต่อวงจรการงอกของเส้นผมและภูมิคุ้มกัน เป้าหมาย 40-70 ng/mL
- **TSH**: ไทรอยด์แฝงทำให้ผมร่วงทั่วศีรษะ (Diffuse Hair Loss)
- **Free Testosterone**: แปลงเป็น DHT ตัวการหลักทำลายรากผม
- **DHEA-S**: แอนโดรเจนจากต่อมหมวกไต เร่ง AGA
- **ANA**: ตรวจ SLE/Autoimmune ที่ทำให้ผมร่วง

## Treatment Recommendations

### Ferritin ต่ำ
แพทย์แนะนำให้รับประทานยาเสริมธาตุเหล็กตามที่จัดให้ เพื่อปรับสมดุลและฟื้นฟูรากผม

### Vitamin D ต่ำมาก (< 20)
ระยะเริ่มต้น (Boost Phase): แนะนำการรักษาด้วยวิตามินดีแบบฉีดเข้ากล้ามเนื้อ (1 เข็ม ทุก 3 เดือน รวม 3 เข็ม) เพื่อให้ร่างกายดูดซึมไปใช้ได้เต็มที่ โดยเฉลี่ย 1 เข็มจะช่วยเพิ่มค่าเลือดได้ประมาณ 10 หน่วย

### DHEA-S สูง + อาการ
หากมีสิว/ประจำเดือนผิดปกติ แนะนำให้ปรึกษาสูตินรีแพทย์

### ANA Positive + อาการ
ปรึกษาแพทย์อายุรกรรมโรคข้อและรูมาติซั่ม หรือแพทย์ผิวหนัง เพื่อควบคุมโรคให้สงบก่อน
